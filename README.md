# 🏦 Rust ATM

**Rust ATM** is a simple command-line banking application that allows users to perform common ATM operations like deposits, withdrawals, and balance checks.

## 🚀 Features
- 💵 Deposit Money
- 💸 Withdraw Money
- 📈 Check Account Balance
- 🔐 User Authentication (Optional)
